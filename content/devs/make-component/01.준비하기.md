 ## 프로젝트 만들기
```bash
pnpm create vite
```
를 통해 React 프로젝트를 만들었다.
여러 옵션이 나오는데 그 중 tailwindCss를 써보려고 했는데, 나중에 프로젝트 할 때 참고하기 위해 만들기 때문에 module별로 명확히 구분이 되는게 좋겠다고 판단이 들어 SCSS를 사용했다.

### 기본 세팅 진행하기
개발 전 작업에 필요한 것들, 전체 영향을 주는 것 부터 고려해서 작업을 진행했다.
style에 공통으로 들어갈 reset css, color 변수, font-size, spacing등을 style에 추가했다.
폴더는 소스를 관리한 다는 생각으로 나눴는데 react는 next.js처럼 폴더 구조가 route가 되지 않기 때문에 폴더를 용도별로 나눴다.

- components
  - component를 관리 하는데 common, layout 등 component들을 관리 하는 용도
- page
  - 말그대로 page 단위 컴포넌트다.
  - url이 folder 이름이 된다.
  - compnent폴더 내에 component의 조합으로 이뤄진다.
  - page내에 서만 쓰는 경우 안에 component 폴더를 만들어 사용할 수 있다.

### 웹폰트
 프리텐다드를 적용했다. 여러 프로젝트를 겪으면서 프리텐다드를 많이 적용했었다.
 다만 다른 점은 이번에는 가변 다이나믹 서브셋(variable Dynamic Subset)을 적용하기로 했다.
 화면에 사용되는 글자만 동적으로 분할하여 다운로드 되기에 초기 페이지 로딩 속도가 빠른 걸로 알고 있기에 이 참에 적용해봤다.(전에는 웹폰트를 그대로 받아서 폴더에 넣어놓고 불러오는 형태로 했다.)

 아래 링크에 자세히 설명되어 있는데 나는 HTML에 `<link>` tag로 적용했다.
 [프리텐다드](https://github.com/orioncactus/pretendard)

 index.html에 `<head>` tag 아래와 같이 적용했다.

 ```html
<link
  rel="stylesheet"
  as="style"
  crossorigin
  href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable-dynamic-subset.min.css" />
 ```
css 에는 font-family만 잘 적용하면 된다.
선언하는 폰트들이 많기도하고 길기 때문에 CSS Variation으로 선언했다.

```css
:root {
  --font-family:
    'Pretendard Variable', Pretendard, -apple-system, BlinkMacSystemFont,
    system-ui, Roboto, 'Helvetica Neue', 'Segoe UI', 'Apple SD Gothic Neo',
    'Noto Sans KR', 'Malgun Gothic', 'Apple Color Emoji', 'Segoe UI Emoji',
    'Segoe UI Symbol', sans-serif;
}

html {
  font-family: var(--font-family);
  line-height: 1.5; /* 1 */
  -webkit-text-size-adjust: 100%; /* 2 */
}
```
